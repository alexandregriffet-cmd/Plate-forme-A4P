# A4P Platform V4

Plateforme statique premium pour GitHub Pages.

## Ce que contient réellement ce pack
- Hub central A4P
- Dashboard club
- Vue équipe
- Vue joueur
- Générateur de passation équipe
- Rapport staff exportable via impression navigateur
- Comptes club et coaches simulés dans localStorage
- Synchronisation locale vers `a4p_hub_results`

## Déploiement
1. Créer un nouveau dépôt GitHub, par exemple `A4P-Platform-V4`.
2. Déposer tout le contenu du pack à la racine du dépôt.
3. Activer GitHub Pages sur la branche principale.
4. Ouvrir `index.html`.

## Limites
- pas d'authentification serveur réelle
- pas de base de données distante
- pas de multi-utilisateurs simultanés sécurisés
